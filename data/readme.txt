The tools in this package expect the following NHTS data tables in csv (at least one year's set).
HHV2PUB (household)
PERV2PUB (person)
VEHV2PUB (vehicle)
DAYV2PUB (trip)
hh50wt (household level weight replicates)
per50wt (person level weight replicates)
